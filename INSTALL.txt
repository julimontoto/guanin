UBUNTU 22.04.1 LTS:

Assuming python 3.9.4 is installed in Ubuntu 22.04.1 LTS

pip or conda will be needed to install dependencies:
pip install:
$ sudo apt update
$ apt install python3-pip
$ pip install --upgrade pip
$ pip install PyQt6
$ python setup.py install
RUN:
$ guanin


MAC/UNIX:

Assuming python ^3.9 is installed.
% pip install python3-pip
% pip install --upgrade pip
% pip install PyQt6
% python3 setup.py install
RUN:
$ guanin
